# 💼 Financial NLP Assistant


## 🚀 Overview

A BloombergGPT-inspired financial NLP system for sentiment analysis, summarization, and question answering using FinBERT, PEGASUS, and FLAN-T5.

## 🔧 Features
- Financial Sentiment Analysis
- Financial Summarization
- Question Answering
- Data Cleaning & Integrity Checks
- Reproducibility Validation
- Batch CSV Processing
- Gradio UI

## ⚙️ Installation
pip install -r requirements.txt

## ▶️ Run
python main.ipynb

## 🌐 UI
python ui/app.py
